# Barcode Generator

এই প্রজেক্টটি EAN13 বারকোড জেনারেট করে এবং পিডিএফ ফাইল হিসেবে এক্সপোর্ট করার সুবিধা প্রদান করে।

## ইনস্টলেশন

1. রিপোজিটরি ক্লোন করুন
2. `npm install` চালান
3. `npm start` চালান

## ব্যবহার

1. ডাটা ফাইল (JSON) আপলোড করুন
2. জেনারেট বাটনে ক্লিক করুন
3. পিডিএফ ডাউনলোড হবে

## টেকনোলজি

- JavaScript
- Webpack
- JsBarcode
- jsPDF
